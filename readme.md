# JS‑Start: сборник учебных заданий по JavaScript


## О проекте

Репозиторий содержит решения базовых учебных задач по JavaScript. Каждая задача вынесена в отдельный каталог, а стартовая страница (`index.html`) предоставляет удобную навигацию между ними.

Проект предназначен для:
- отработки базовых конструкций JavaScript;
- изучения взаимодействия JS с DOM;
- практики работы с событиями браузера;
- формирования структуры учебного репозитория.

## Структура проекта

├── index.html # стартовая страница с навигацией
├── task1/ # Задача 1: вывод в консоль
│ ├── index.html
│ └── script.js
├── task2/ # Задача 2: обработчики для console.log, alert, prompt
│ ├── index.html
│ └── script.js
├── task3/ # Задача 3: светофор с цикличной сменой цветов
│ ├── index.html

│ └── script.js
├── task4/ # Задача 4: изменение текста ссылки через prompt
│ ├── index.html
│ └── script.js
└── task5/ # Задача 5: перехват формы и дублирование ввода
├── index.html
